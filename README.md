## Description

KFUPods is a smart nap pod system designed for the **KFUPM campus**, providing:
- A mobile app for booking and controlling pods.
- A web-based prototype for visualizing pod locations and managing sessions.

This project was developed as a course requirement and demonstration prototype, not as a production system.
